About


Sometime back I along with a bunch of other guys did a freelance project, building a website for one of the small companies. Since we were geographically distributed, we shared our temporary code in pastebin and commented on each of our code. But guess what we did for UI? We took a snapshot of the temporary UI we designed and shared it in our mailinglist asking for feedback.We also mailed the UI snapshot to our clients and asked their feedback. Thinking of improving this process, I ended up with this site. Hope you too love it!


If you got any other better method to share design or any improvements for the site, please do write to me at aravindkumar [dot] leo [at] gmail.com


This site has been developed using bottle.py + jQuery File Upload + twitter bootstrap.Am thankful to these awesome projects.The site has been deployed in amazon aws.
